# SIB Internal Web Portal

Internal web portal for Sridhar Insurance Broker Pvt. Ltd.

## About
A frontend web portal built for employees to manage policies, clients, and claims digitally.

#Pages Built
- Dashboard  
- Analytics  
- Policies  
- Clients  
- Claims  
- UI Components  
- My Tasks  
- Documents  
- Settings  
- API / Backend Simulation  

## Technologies Used
-  **HTML5** – Semantic structure and layout  
- **CSS3** – Flexbox, Grid, Animations, CSS Variables, Dark Mode  
- **JavaScript (ES6+)** – DOM Manipulation, Event Handling, Dynamic UI
